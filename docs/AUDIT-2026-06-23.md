## Architecture (hexagonal)

```mermaid
flowchart TB
  subgraph ui [UI src/ui + src/data]
    ExpensesApp --> resolveSource
    resolveSource --> csvDS[csvDataSource dev]
    resolveSource --> apiDS[apiDataSource prod]
    useExpenseActions --> datasetPatches
  end

  subgraph domain [Domain src/domain]
    subgraph engine [Engine pure]
      status
      recurring
      budgets
      cashRecon[cash recon]
    end
    subgraph ports [Ports]
      ExpenseRepository
      AccessRepository
    end
    ExpenseDataSource[ExpenseDataSource client port]
    engine --> types
  end

  subgraph workers [Workers functions/]
    Middleware --> API
    API --> ExpenseRepository
    Adapters --> ExpenseRepository
    Adapters --> D1[(D1)]
  end

  apiDS --> API
  functionsDomain["functions/domain symlink"] -.-> domain
```

| Check | Result |
